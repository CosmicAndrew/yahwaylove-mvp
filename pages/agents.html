The 9-Step Heartbeat Cycle

1

Wake

Agent activates from sleep state on cron trigger or webhook event

2

Identity

Retrieves agent profile from agents/ folder (director.md, aria.md, etc.)

3

Governance

Checks budget status — soft alert at 80%, hard stop at 100%

4

Continuation

Loads prior context from wiki/ folder and continuation pointer

5

Adapter

Resolves which AI (Claude Code, Codex, OSA) handles this task type

6

Tasks

Fetches pending tasks from Supabase with atomic checkout (no double-execution)

7

Execute

Runs task with appropriate specialist agent and tool calls

8

Compact

Forks second Claude to summarize session, re-injects compressed context

9

Sleep

Writes outputs/ and wiki/ updates, schedules next wake interval

Advanced Architecture

Claude Code Internals Unlocked

Three layers of Claude Code infrastructure that most developers don't know exist — all active inside YAHWAYLOVE's agent system.

KAIROS Daemon

Anthropic's internal autonomous agent pattern discovered via Claude Code source analysis by Sabrina Ramonov (March 2026). GitHub webhook listener triggers on push events. 5-minute cron scheduler maintains continuous heartbeat. /dream endpoint consolidates memory during idle periods.

UltraPlan

Activated via free-code's ULTRAPLAN flag, UltraPlan offloads complex planning to a remote Claude Opus instance with a 30-minute session window and 3-second polling. Enables deep multi-step planning that the standard model context can't hold.

claude-opus-4-5 + budget_tokens: 10000

free-code (54 Flags)

The free-code fork removes all telemetry and enables 54 experimental flags. Key flags: ULTRAPLAN, ULTRATHINK (extended reasoning), AGENT_TRIGGERS (autonomous task triggering), TEAMMEM (shared memory between agents), VOICE_MODE, VERIFICATION_AGENT, BRIDGE_MODE.
