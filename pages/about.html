AI Training Camp Curriculum

Andrew's Signature Tools

Every tool Andrew teaches in the AI Training Camp is a tool he uses daily to deliver client results. No filler. No theory-only content. These are the exact instruments behind every campaign, every website, every automation YAHWAYLOVE builds.

Claude Code

Andrew's primary development environment. Builds every agent system, every client website, every automation workflow. The BUILD agent runs entirely inside Claude Code sessions connected to GitHub and Vercel.

Anthropic API

The backbone of all 10 agents. Andrew builds each agent using Claude's tool-use API, enabling multi-step autonomous execution, real-time research, code generation, and cross-agent coordination on client campaigns.

Prompt Engineering

The core craft behind every agent. Andrew has developed proprietary prompt systems — including the Sony A1 Enhancement framework — that give agents consistent voice, precision, and context memory across client accounts.

Google Gemini

Used for research cross-referencing and content ideation. SCOUT and CONTENT agents use Gemini for long-context analysis, multi-document synthesis, and competitive landscape comparisons that benefit from a second model's perspective.

NotebookLM

Andrew's knowledge management backbone. Used to upload sermon transcripts, client strategy documents, and research sources — then query them conversationally. Powers the sermon-to-social feature that CONTENT agent uses for church clients.

GitHub

Version control for every BUILD agent output. Every client website, every automation script, every landing page is committed to a GitHub repo — meaning full version history, rollback capability, and clean code handoff on project completion.

JSON Context Profiles

Custom memory and persona files for each agent and each client. These JSON profiles give agents persistent context — brand voice, audience profile, past campaign data, theological convictions — so they never start from scratch on client work.

Perplexity AI

Real-time research engine powering SCOUT and ARIA. Used daily to track competitor ad activity, identify trending faith content, monitor algorithm changes on Meta, and pull fresh market data for campaign briefs.

VS Code

Primary IDE for all Claude Code workflows. Andrew runs Claude Code extensions inside VS Code for AI-assisted development, integrating directly with GitHub for version control and Vercel for instant deployment on every BUILD project.

MCP Connectors

Anthropic's Model Context Protocol — the bridge between Claude agents and the real world. Andrew uses MCP to connect agents directly to Meta Ads, HubSpot, Twilio, Google Analytics, GitHub, and Vercel. 110M+ monthly downloads. Enterprise power, Kingdom price.
