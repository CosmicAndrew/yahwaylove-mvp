The Curriculum

What You'll Build

5 modules. Every lesson is hands-on. You leave with working systems, not just slides.

Module 01
Foundation — Claude Code & Agent Thinking
Claude Code Openclaw VS Code
Install your development environment. Write your first Claude Code agent. Understand tool use, system prompts, and the mindset shift from "AI tool user" to "AI systems builder." Build a working ministry FAQ bot by end of session.
Module 02
Prompt Engineering Mastery
Prompt Engineering Claude Cowork JSON Profiles
Andrew's complete prompt engineering system — including the Sony A1 portrait enhancement framework, chain-of-thought prompting, role-based personas, and JSON Context Profiles for persistent agent memory. Build a reusable prompt library for your ministry.
Module 03
Research & Intelligence Systems
Perplexity Computer Google Gemini NotebookLM
Build an AI research pipeline that monitors your ministry's landscape, summarizes sermon resources, tracks competitor churches, and generates weekly intelligence briefings — automatically. Use NotebookLM to create an AI-powered knowledge base from your sermon library.
Module 04
Build & Deploy with GitHub
GitHub Claude Code Vercel
Deploy a real ministry landing page using Claude Code and GitHub — no prior coding experience required. Understand version control, auto-deployment to Vercel, and how to maintain and update AI-built websites. Leave with a live URL.
Module 05
MCP & Agent Orchestration — Connecting Everything
MCP Anthropic API HubSpot Twilio
The capstone. Use Model Context Protocol (MCP) to connect your Claude agents to HubSpot, Google Calendar, your website, and email — so your systems talk to each other automatically. Build a simple DIRECTOR-style orchestrator that routes tasks between two specialized agents. This is the same architecture Andrew uses in production for YAHWAYLOVE clients.
